学部1年次に作成したアプリです。実際に日常生活で使用できるアプリケーションを作成するという講義目標に従って作成しました。
何かしらの目標時間を設定すると、達成率ごとに3Dモデルの島が自然豊かになっていきます。

## 実行
```App_wip```からアプリを起動します

## 機能
* 目標時間の設定
* 達成時間の入力
* 初期化
* 達成率の確認
